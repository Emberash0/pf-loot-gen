# Pathfinder First Edition Loot Generator

The loot generator accepts varied arguments:
- Run without arguments and it will guide you through an encounter building process.
- Run with only a total quantity of creatures, this will automatically continue once the encounter contains the same number of creatures.
- Run with just a creature name. It will ask for a quantity and then auto-continue
- Run with a creature name before or after a quantity. It will process an encounter consisting of a single creature type of the given quantity.

The loot generator will then automatically assign treasure types and randomly generate treasure. Where a creature always leaves certain items, these will be guaranteed to drop. The loot generator also gives the total XP award for the encounter
